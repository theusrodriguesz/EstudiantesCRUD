# Estudiantes CRUD

Este proyecto implementa un sistema CRUD (Crear, Leer, Actualizar, Eliminar) para gestionar estudiantes y sus asignaturas.

## Funcionalidades

1. **Crear estudiante**: El sistema permite agregar estudiantes con un ID único, nombre completo, edad, fecha de nacimiento y las asignaturas en las que están matriculados.
2. **Listar estudiantes**: Muestra una lista de todos los estudiantes con las asignaturas y sus notas.
3. **Actualizar notas**: Permite modificar la nota de una asignatura de un estudiante.
4. **Eliminar estudiante**: Elimina un estudiante y sus asignaturas.
